各区域说明

区域位置功能
组件面板左侧栏展示所有19个可用组件，分为"输入型组件"（4个）、"选择型组件"（13个）、"布局型组件"（2个）三大类。点击或拖拽即可将组件添加到画布。
画布区域中间主区域表单的可视化编辑区域。顶部有"表单模板名称"输入框和"清空"、"提交表单"按钮。组件添加后可点选查看属性，鼠标悬浮可复制或删除组件。
属性面板右侧栏包含两个Tab页："组件属性"用于配置当前选中组件的各项参数，"表单属性"用于配置整个表单的全局设置。

二、快速入门教程

按照以下步骤，你可以在5分钟内创建你的第一个表单模板。

登录管理后台

访问后台地址，使用管理员账号登录系统。输入验证码后点击登录。

进入表单设计器

在左侧菜单找到 活动管理 > 活动表单模板，点击进入模板列表页面，然后点击 "新增模板" 按钮进入表单设计器。

填写表单模板名称

在画布顶部的输入框中输入模板名称，例如"活动报名表"。这是必填项，提交前必须填写。

添加组件到画布

从左侧组件面板中，点击或拖拽你需要的组件到中间画布区域。组件会按添加顺序从上到下排列。

配置组件属性

点击画布中的某个组件，右侧属性面板会显示该组件的所有可配置项。你可以修改标题、字段名、设置必填、添加校验规则等。

配置表单属性（可选）

点击右侧属性面板的 "表单属性" Tab，可以设置全局的标签对齐方式、表单尺寸、栅格间隔等。

提交保存模板

确认无误后，点击画布顶部的 "提交表单" 按钮。系统提示"新增成功"即表示模板已保存到后端数据库。

小提示

  • 复制组件：鼠标悬浮在画布中的组件上，点击右上角的"复制"图标可快速复制该组件。
  • 删除组件：点击组件右上角的"删除"图标可移除该组件。
  • 清空画布：点击"清空"按钮可一键清除所有已添加的组件，重新开始设计。
  • 字段名唯一：每个组件的"字段名"（v-model）在表单中必须唯一，系统会自动生成默认字段名。

单行文本

输入型 ▶
用途：用于收集一行简短文字信息，如姓名、手机号、邮箱等。支持前后缀、图标、字数限制、正则校验等丰富配置。
使用模板： 活动报名表 志愿者申请表 素材提交表
属性名称类型默认值说明
组件类型下拉选择单行文本定义该组件的渲染类型，不可更改为其他大类。
字段名文本field10X绑定到表单数据模型(v-model)的字段标识，须在表单内唯一。例如 userName、mobile。
标题文本单行文本显示在组件左侧（或上方）的标签文字，如"姓名"、"手机号"。
占位提示文本请输入XXX输入框为空时显示的灰色提示文字，引导用户输入。
表单栅格滑块 1-2424基于24栅格系统控制组件占用的宽度。24=整行，12=半行，8=三分之一行。
标签宽度数字跟随表单覆盖表单全局的标签宽度设置，单位px。留空则跟随表单属性中的设置。
组件宽度文本100%输入框本身的宽度，支持百分比和像素值。如 100% 或 300px。
默认值文本空组件的初始填充值，表单渲染后即显示此值。
前缀文本空在输入框前方显示的文字，如"http://"。
后缀文本空在输入框后方显示的文字，如"元"、"人"。
前图标图标选择空在输入框前方显示的Element UI图标，如 el-icon-mobile（手机图标）、el-icon-search（搜索图标）。可手动输入图标类名或点击"选择"按钮从图标库中选取。
后图标图标选择空在输入框后方显示的图标，用法同前图标。
最多输入数字空（不限制）限制用户可输入的最大字符数。如手机号设为11，身份证设为18。
输入统计开关关闭开启后在输入框右下角显示已输入/最大字符数（如"5/11"），需配合"最多输入"使用。
能否清空开关开启开启后输入框有内容时，右侧显示清空按钮(x)，一键清除输入。
是否只读开关关闭开启后输入框可显示内容但不可编辑，适合展示预填数据。
是否禁用开关关闭开启后组件变为灰色不可操作状态。
是否必填开关开启开启后标题前显示红色星号(*)，提交时检查是否为空。
正则校验规则组空点击"添加规则"可设置正则表达式校验。每条规则包含：
表达式：正则表达式，如 /^1(3|4|5|7|8|9)\d{9}$/（手机号）
错误提示：校验不通过时的提示文字，如"手机号格式错误"。
使用场景举例：在"活动报名表"模板中，"手机号"字段使用了单行文本组件，配置了前图标 el-icon-mobile、最多输入11个字符、开启输入统计、并添加了手机号正则校验规则。

五、表单属性

点击右侧属性面板的"表单属性"Tab，可以配置整个表单的全局设置。这些设置影响表单中所有组件的默认表现。

表单名
ref: elForm
表单组件的引用名称，用于代码中获取表单实例进行校验等操作。默认值 elForm。
表单模型
model: formData
表单数据绑定的对象名称，所有字段值都存储在此对象中。默认值 formData。
校验模型
rules: rules
表单校验规则的对象名称，包含所有必填和正则校验规则。默认值 rules。
表单尺寸
size
统一设置所有组件的尺寸大小。
中等（默认） 较小 迷你
标签对齐
label-position
所有组件标签（标题）的对齐方式。
左对齐 右对齐 顶部对齐（默认）
标签宽度
label-width: 100px
所有标签的统一宽度，单位px。数字越大，标签占的空间越多。仅在"左对齐"和"右对齐"时生效。默认100px。
栅格间隔
gutter: 15
表单栅格布局中列与列之间的间距，单位px。默认15px。
禁用表单
disabled: false
开启后整个表单变为只读禁用状态，所有组件不可操作。适合查看模式。
表单按钮
formBtns: true
是否显示表单底部的"提交"和"重置"按钮。默认开启。关闭后可使用自定义按钮组件代替。
显示未选中组件边框
unFocusedComponentBorder
在设计器中是否给未选中的组件显示边框轮廓。仅影响设计时的预览效果，不影响最终表单。
