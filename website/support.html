常见问题解答

账户与注册

如何注册西柚-pomelo账户？

西柚-pomelo采用推荐码注册系统。您需要：
1. 获得有效的推荐码（通过组织管理员或活动发布方）
2. 使用西柚-pomelo应用扫描推荐码二维码
3. 填写注册信息：法定姓名、英文昵称、学校邮箱等
4. 提供中国或美国电话号码进行验证
5. 完成邮箱验证和手机验证

为什么需要推荐码才能注册？

推荐码系统确保只有真正的海外中国学生组织成员可以使用我们的服务， 这有助于维护社区的安全性和真实性，也方便组织管理员更好地管理成员。

忘记密码怎么办？

目前密码重置功能正在开发中。如果您忘记密码，请发邮件至 info@americanpromotioncompany.com， 我们的支持团队会协助您重置密码。

活动与报名

如何报名参加活动？

1. 在活动列表中浏览感兴趣的活动
2. 点击活动卡片查看详细信息
3. 点击"报名"按钮（需要先登录）
4. 填写动态报名表单
5. 提交后获得报名确认和二维码

报名成功后如何参加活动？

报名成功后，您会收到一个唯一的二维码。在活动现场， 组织方会扫描您的二维码进行核销确认。请保存好这个二维码， 建议截图保存到手机相册中。

可以取消报名吗？

目前报名取消功能正在开发中。如果您需要取消报名， 请联系活动组织方或发邮件给我们的支持团队。

志愿者服务

如何记录志愿服务时长？

志愿服务时长由组织管理员通过专用的签到系统记录：
1. 到达志愿服务地点后，向管理员提供您的手机号
2. 管理员为您办理签到，记录开始时间
3. 志愿服务结束后办理签出，记录结束时间
4. 系统自动计算本次志愿时长并累加到您的总时长

志愿时长数据可以用于申请总统奖吗？

是的！我们的志愿时长记录系统严格按照官方要求设计， 记录的数据可以作为总统志愿服务奖等荣誉申请的支持材料。 您可以随时导出官方格式的志愿服务证明。

技术问题

应用无法正常加载怎么办？

请尝试以下解决方法：
1. 检查网络连接是否正常
2. 重启应用程序
3. 更新到最新版本
4. 重启手机
如果问题仍然存在，请联系技术支持并说明具体情况。

二维码扫描不成功怎么办？

1. 确保手机摄像头权限已开启
2. 在光线充足的环境下扫描
3. 保持手机稳定，对准二维码
4. 清洁手机摄像头镜头
5. 如果二维码模糊，请联系提供方重新获取

支持哪些设备和系统版本？

西柚-pomelo 支持：
• iOS：iOS 13.0 及以上版本
• Android：Android 8.0 (API 26) 及以上版本
建议使用最新版本的操作系统以获得最佳体验。

我的数据安全吗？
