安装配置

安装 Python 依赖

项目依赖 httpx、beautifulsoup4、lxml、sqlalchemy 等库。

安装 Playwright 浏览器（可选，用于 JS 渲染文章）

部分公众号文章需要浏览器渲染才能获取完整内容。

安装 PyWxDump（可选，用于聊天记录自动解密）

仅 Windows 平台可用，自动提取微信数据库密钥。

# 1. 安装核心依赖
cd crawler/wechat
pip install httpx beautifulsoup4 lxml sqlalchemy[asyncio] asyncpg python-dotenv

# 2. 安装 Playwright（可选 — JS 渲染回退）
pip install playwright
playwright install chromium

# 3. 安装 PyWxDump（可选 — Windows 自动密钥提取）
pip install pywxdump

# 4. 安装 pysqlcipher3（可选 — 聊天记录解密）
pip install pysqlcipher3
环境要求 Python 3.10+，推荐 3.11+。Playwright chromium 约占 400MB 磁盘空间。

快速上手

最简单的方式是直接提供文章 URL 进行爬取：

python scripts/crawl_articles.py \
  --urls "https://mp.weixin.qq.com/s/xxxxxxx" \
  --output-dir ./output/articles \
  --no-db

爬取结果保存在 ./output/articles/ 目录，每篇文章一个 JSON 文件，外加一个 index.json 索引。

方式 A — 直接 URL 爬取

直接 URL — 推荐方式

成功率 ~90%

直接提供一个或多个 mp.weixin.qq.com 文章链接，爬虫会获取文章全文、图片、视频等。

适用场景：已知文章链接、从朋友圈/群聊获取的分享链接、需要精确爬取特定文章。

单篇文章

python scripts/crawl_articles.py \
  --urls "https://mp.weixin.qq.com/s/xxxxxxx"

多篇文章

python scripts/crawl_articles.py \
  --urls "https://mp.weixin.qq.com/s/aaa" "https://mp.weixin.qq.com/s/bbb" \
  --download-media \
  --output-dir ./output/articles

爬取结果字段

字段说明示例
title文章标题"2025年留学新政策"
author作者"小编"
account_name公众号名称"留学全知道"
content纯文本正文"近日，教育部..."
publish_date发布日期"2025-01-15"
images图片 URL 列表["https://mmbiz..."]
source_url原始链接"https://mp.weixin..."
reading_count阅读数0 (永远为0)

方式 B — 搜狗搜索

搜狗微信搜索

成功率 ~60%

通过搜狗微信搜索（weixin.sogou.com）按关键词或公众号名称搜索文章，然后逐篇爬取。

注意 搜狗反爬较严格，单 IP 发送几十次请求即触发验证码。建议配合代理池或打码平台使用。

按关键词搜索

python scripts/crawl_articles.py \
  --keyword "留学签证" \
  --max-articles 20 \
  --max-pages 3

按公众号名称搜索

python scripts/crawl_articles.py \
  --account "留学全知道" \
  --max-articles 50

使用代理

python scripts/crawl_articles.py \
  --keyword "留学" \
  --proxy "http://127.0.0.1:7890"

使用 Sogou Cookies（延长会话）

# 先在浏览器登录搜狗，导出 cookies 为 JSON
python scripts/crawl_articles.py \
  --keyword "签证" \
  --sogou-cookies sogou_cookies.json

方式 C — 历史 API

微信内部历史 API

成功率 ~55%

通过微信内部 API（mp.weixin.qq.com/mp/profile_ext）拉取公众号的全部历史文章。需要有效的微信 Cookie。

前提条件 需要通过 mitmproxy 或手动抓包获取微信客户端的 Cookie，Cookie 有效期通常只有 2~24 小时。

基本用法

python scripts/crawl_articles.py \
  --biz "MzA3MDk1NDkxMQ==" \
  --history \
  --wechat-cookies cookies.json \
  --max-articles 100

Cookie 文件格式

// cookies.json
{
  "wxuin": "123456789",
  "wxsid": "abcdef...",
  "pass_ticket": "xxxxx...",
  "wap_sid2": "xxxxx..."
}

获取 __biz 参数

在微信中打开公众号任意文章，查看 URL 中的 __biz 参数（Base64 编码字符串）。

断点续传

如果爬取中断（网络超时、Cookie 过期等），爬虫会自动保存 checkpoint。再次运行相同命令会从上次断点继续。

三种方式对比

特性方式 A — 直接 URL方式 B — 搜狗搜索方式 C — 历史 API
成功率 ~90% ~60% ~55%
需要 Cookie否可选（延长会话）是（必须）
需要代理否建议否
文章数量手动指定搜索结果数全部历史
反爬强度低高（验证码）中（频率限制）
推荐场景已知链接按主题发现文章全量采集某账号

聊天记录提取

从微信本地数据库（EnMicroMsg.db）中提取聊天记录，支持解密、过滤和多格式导出。

平台支持 微信本地数据库仅存在于 Windows 和 Android 平台。macOS/iOS 版微信不使用 SQLite 存储聊天记录。

数据库解密

方法 1：PyWxDump 自动提取密钥（Windows）

PyWxDump 可以自动从运行中的微信进程提取数据库密钥。

# 确保微信正在运行
pip install pywxdump
pywxdump bias  # 自动获取偏移地址和密钥

方法 2：手动提供密钥

python scripts/extract_chat.py \
  --db-path ./EnMicroMsg.db \
  --key "abc123def" \
  --output-format json

过滤联系人和日期范围

python scripts/extract_chat.py \
  --db-path ./EnMicroMsg.db \
  --key "abc123def" \
  --contacts "wxid_123,wxid_456" \
  --date-range "2024-01-01:2024-12-31" \
  --output-format csv

导出格式

格式参数说明适用场景
JSON --output-format json 结构化 JSON，包含所有字段 程序处理、数据分析

TXT --output-format txt 可读文本，按对话分组 人工阅读、存档

Python API

除了 CLI 命令行，所有功能也可以通过 Python API 直接调用。

文章爬取 API

import asyncio
from src.official_account.crawler import (
    ArticleCrawler,
    SogouWeixinCrawler,
    HistoryCrawler,
)

async def main():
    # ---- 直接爬取单篇文章 ----
    crawler = ArticleCrawler()
    article = await crawler.crawl_article(
        "https://mp.weixin.qq.com/s/xxxxxxx"
    )
    print(article.title, article.content_text[:100])
    await crawler.close()

    # ---- 批量爬取多篇文章 ----
    crawler = ArticleCrawler(download_media=True, media_dir="./media")
    urls = ["https://mp.weixin.qq.com/s/aaa", "https://mp.weixin.qq.com/s/bbb"]
    articles = await crawler.crawl_multiple(urls, concurrency=3)
    await crawler.close()

    # ---- 搜狗关键词搜索 ----
    sogou = SogouWeixinCrawler(proxy="http://127.0.0.1:7890")
    results = await sogou.search_articles("留学签证", max_pages=2)
    for r in results:
        print(r.title, r.url)
    await sogou.close()

asyncio.run(main())

数据库存储

爬取结果可以存入 PostgreSQL 数据库，使用 SQLAlchemy ORM 模型。默认同时导出为本地 JSON 文件。

启用数据库存储

# 通过环境变量
export DATABASE_URL="postgresql://user:pass@localhost:5432/wechat_db"
python scripts/crawl_articles.py --keyword "留学"

# 或通过命令行参数
python scripts/crawl_articles.py \
  --keyword "留学" \
  --database-url "postgresql://user:pass@localhost:5432/wechat_db"

# 跳过数据库，仅导出文件
python scripts/crawl_articles.py --keyword "留学" --no-db

数据库表结构

表名用途关键字段
wechat_articles 公众号文章 title, content, source_url (唯一键), publish_date, images (JSONB)
wechat_official_accounts 公众号账号注册 biz_id (唯一键), name, account_type, last_crawled_at
wechat_messages 聊天消息 conversation_id, sender_name, message_type, content, timestamp
wechat_contacts 联系人 wechat_id (唯一键), nickname, contact_type
去重机制 文章以 source_url 作为唯一键，重复爬取会自动更新（upsert）而非重复插入。

验证码与代理

验证码打码服务

搜狗搜索触发验证码后，爬虫会自动调用打码服务。支持两种服务：

服务环境变量说明
2Captcha（国际） CAPTCHA_SERVICE=2captcha
CAPTCHA_API_KEY=xxx		 英文平台，按次计费，$2.99/1000次
超级鹰（国内） CAPTCHA_SERVICE=chaojiying
CHAOJIYING_USER=xxx
CHAOJIYING_PASS=xxx		 中文平台，按点数计费

代理池配置

支持三种代理配置方式（按优先级）：

# 方式 1：代理池 API（每次请求获取新 IP）
export PROXY_POOL_URL="http://your-proxy-api/get"

# 方式 2：静态代理列表
export PROXY_LIST="http://user:pass@ip1:port,socks5://ip2:port"

# 方式 3：proxies.txt 文件（每行一个代理地址）
# 放在 crawler/wechat/proxies.txt
http://user:pass@ip1:port
socks5://ip2:port
自动容错 连续 3 次触发验证码后，爬虫会自动切换代理并刷新 Cookie。失败的代理会被从池中移除。

常见问题

Q: 搜狗搜索总是触发验证码怎么办？

A: 配置代理池（PROXY_LIST 或 PROXY_POOL_URL）并启用打码服务。也可以先在浏览器手动访问搜狗获取 Cookie，通过 --sogou-cookies 传入。

Q: 历史 API 返回 "ret=-1" 或空数据？

A: Cookie 已过期。需要重新通过 mitmproxy 抓包获取微信客户端的新 Cookie。Cookie 有效期通常 2~24 小时。

Q: 文章内容为空或不完整？

A: 该文章可能需要 JS 渲染。确保已安装 Playwright（pip install playwright && playwright install chromium），爬虫会自动回退到浏览器渲染。

Q: pysqlcipher3 安装失败？

A: 该库需要 SQLCipher C 库。Ubuntu: apt install libsqlcipher-dev。macOS: brew install sqlcipher。Windows: 使用预编译 wheel 或通过 conda 安装。

Q: 如何获取公众号的 __biz 参数？

A: 在微信中打开该公众号的任意文章，复制链接。URL 中 __biz= 后面的值即为 biz 参数（Base64 字符串，通常以 == 结尾）。
